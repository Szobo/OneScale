# OneScale Frontend

OneScale is an AI-native data platform that turns operational data into real-time
intelligence. The frontend showcases the OneScale product experience — from data
ingestion and analytics to AI insights and user journeys — and provides a
waitlist entry point for early access.

## What This UI Covers

- Landing page with waitlist call-to-action and product overview.
- Product pages for data ingestion, analytics dashboards, and AI insights.
- User journey storytelling that walks through the end-to-end platform flow.
- Authenticated routes for dashboard and file uploads.

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router

## Local Development

```bash
npm install
npm run dev
```
